# Kryten Assist

> **Making Future Robin's Life Easier**

Kryten Assist is a long-term personal digital assistant built as both a learning project and a portfolio.

The project follows Clean Architecture principles and aims to explore modern software engineering practices, AI-assisted development, and cross-platform application development.

## Vision

Build software that reduces friction between an idea and getting it done.

Every feature should answer one question:

> **Does this make Future Robin's life easier?**

## Technology

- .NET
- Clean Architecture
- Web API
- React (planned)
- Avalonia (planned)
- AI-assisted development

## Development Philosophy

The project is built using an AI Playbook where every significant feature begins with a documented prompt, followed by implementation, review and learning.
